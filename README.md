# homebrew-tunnel-manager

Homebrew tap for [Tunnel Manager](https://github.com/Goodmain/tunnel-manager) — a macOS menu bar app for AWS SSM port-forwarding tunnels.

## Install

```bash
brew tap Goodmain/tunnel-manager
brew install --cask tunnel-manager
```

## Upgrade

```bash
brew upgrade --cask tunnel-manager
```

## Uninstall

```bash
brew uninstall --cask tunnel-manager
brew untap Goodmain/tunnel-manager
```

## Requirements

Installed automatically as dependencies: `aws-vault`, `awscli`, `session-manager-plugin`.
